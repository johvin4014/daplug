# DaPlug

DaPlug is a map-based event discovery platform where users can find events near them, RSVP or buy tickets, and connect with hosts, venues, and other attendees.

## MVP Goals
- Auth (signup/login)
- Map with event pins
- Create & view events
- RSVP for free events
- Basic host tools (manage your events)

## Tech Stack (MVP)
- Expo (React Native)
- Supabase (Postgres, Auth, Storage)
- Mapbox (maps)
- Stripe (Phase 2)
