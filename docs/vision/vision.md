# Vision

DaPlug is a map-based event discovery platform where users can find events near them, RSVP or buy tickets, and connect with hosts, venues, and other attendees.

## Who it’s for
- Guests: discover events, RSVP, share, and connect
- Hosts/Promoters: create events, manage attendees, promote events
- Venues/Businesses: run recurring events and see performance
- Admins: protect the community through moderation and safety tools

## MVP goal
Ship a working mobile MVP where users can:
1) sign up / log in
2) see nearby events on a map
3) view event details
4) create events (host flow)
5) RSVP to free events
