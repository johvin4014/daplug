# MVP Requirements

## Must Have (MVP)
### Auth
- Users can sign up / log in / log out
- Users can edit a basic profile (name, photo optional)

### Events
- Host can create an event:
  - title, description, category, date/time, location, image
- Users can view event list and event details
- Users can RSVP to free events

### Map
- Map view shows event pins
- Tapping a pin opens event preview/details

### Basic Moderation
- Users can report an event (simple report reason)

## Nice to Have (Post-MVP)
- Paid tickets
- QR check-in
- DMs
- Verified hosts
- Promoted events
